# Coin App

Минималистичное приложение для подбрасывания монетки на базе **PixiJS** с использованием **TypeScript** и **GSAP** для анимаций.

coin-app/
├─ src/
│  ├─ core/          # Базовые абстрактные классы и утилиты
│  ├─ game/          # Логика игры, медиаторы
│  ├─ assets/        # Текстуры, спрайты, звуки
│  └─ main.ts        # Точка входа
├─ package.json
├─ tsconfig.json
├─ vite.config.ts
└─ README.md
